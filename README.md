## Getting Started

### Prerequisites

- [Flutter SDK](https://docs.flutter.dev/get-started/install) (3.0 or higher)
- Dart SDK (included with Flutter)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/mrkweko/todo_app.git
   cd todo_app
   ```

2. Install dependencies:
   ```bash
   flutter pub get
   ```

3. Run the app:
   ```bash
   flutter run
   ```

## Usage

- **Add a task**: Type in the text field and tap the add button
- **Complete a task**: Tap the checkbox next to the task
- **Edit a task**: Long press or use the edit icon on a task
- **Delete a task**: Swipe or use the delete icon
- **View history**: Open the drawer menu and tap "History"
- **Change theme**: Go to Settings in the drawer menu

## Tech Stack

- **Framework**: Flutter
- **State Management**: StatefulWidget
- **Storage**: SharedPreferences
- **Design**: Material Design 3
